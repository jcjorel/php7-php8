
To test a migration from PHP7 to PHP8, create a simple PHP7 application using known incompabilities with PHP8. 
You will test it with a browser and a php7 docker container. Use "docker compose, not "docker-compose".

Tu réaliseras une application de test de type single web page, sans besoin de clic, contenant une liste de tests de chaque incompatibilité PHP7/PHP8 connue. Tu vérifieras avec le serveur MCP Perplexity que tes tests sont bons.
Cette application sera exécutée en parallele dans 2 containeurs via docker compose.

A la fin de la tache, tu créeras dans un répertoire scratchpad/, un plan d'implémentation détaillé, découpé en phases implémentables indépendament, avec gestion de progression de la création de la documentation et de l'implémentaiton du plan. Ce plan d'éxécution sera éxécuté dans une autre session Cline donc termine ta tache dés que tu as créé le plan.
